삼지 마세요. 최신 정보는 [`docs/00-overview/`](../00-overview/)에서 시작합니다.

코드와 어긋날 수 있습니다. 이력 추적 목적으로만 보존합니다.

(`docs/80-reference/81-docs-conventions.md`, gzh-cli-devbox)의 90-99 슬롯

없습니다.
