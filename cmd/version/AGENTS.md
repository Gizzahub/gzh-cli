## 🎯 모듈 특성

**version**은 애플리케이션 버전 정보를 표시하는 단순한 모듈입니다.

### 핵심 기능
- 현재 버전 표시
- 빌드 정보 출력
- Git 커밋 해시 표시

## ⚠️ 개발 시 주의사항

- **빌드 정보 임베딩**: 컴파일 시점에 버전 정보 주입
- **포맷 일관성**: JSON, 텍스트 등 출력 형식 통일
- **의존성 최소화**: 다른 모듈에 대한 의존성 없이 독립 실행

**핵심**: 가장 단순한 모듈이지만 빌드 시스템과의 연동이 중요합니다.
